Downloads

LeoCAD is very easy to install and it's available for all major Operating Systems. In the sections below you'll find detailed download and installation instructions for each OS.

Windows

Click the button below to download the latest version of the LeoCAD installer. Once the download is complete run the installer and follow the installation instructions.

Download LeoCAD for Windows

Linux

LeoCAD is available directly from several distributions, so you may want to search for it in your package manager first. If you can't find it, you'll have to download the source code and compile your own binary.

If you don't have the LDraw Parts Library installed you'll need to download it and extract the zip file to /usr/share/leocad.
